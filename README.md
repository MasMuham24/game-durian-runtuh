Game Durian Runtuh adalah mini game berbasis web yang dibuat menggunakan HTML, CSS, dan JavaScript.  
Pemain harus menghindari atau menangkap durian yang jatuh untuk mendapatkan skor sebanyak mungkin sebelum waktu habis.

Project ini dibuat sebagai latihan pengembangan logika pemrograman, manipulasi DOM, serta implementasi sistem game sederhana berbasis browser.
